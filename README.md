# QSL Card Manager

## 简介

QSL Card Manager 是一款专为业余无线电爱好者设计的桌面应用程序，用于高效管理通联日志（QSO）和生成QSL卡片标签。本软件使用 PyQt5 构建图形用户界面，并利用 SQLite 数据库进行数据存储，确保了操作的便捷性和数据的可靠性。

## 主要功能

- **日志管理**: 提供添加、编辑、删除和查看通联日志的完整功能。
  
- **ADIF 支持**: 支持导入和导出标准的 ADIF (`.adi`) 文件，方便与其他日志软件交换数据。
  
- **QSL 卡片生成**: 可为单条或多条日志生成两种版式的QSL卡片标签，并导出为高分辨率的 PNG 图片。
  
- **NFC 卡写入**: 支持通过串口将QSL卡号写入NFC卡片，方便进行物理追踪和管理。
  
- **智能过滤与搜索**: 用户可以根据我方呼号、对方呼号、通联模式或QSL卡号等多种条件快速筛选和查找日志。
  
- **数据维护**: 内置重复日志检查与合并功能，并提供一键重置QSL数据及按时间重新排序日志的选项。
  
- **呼号管理**: 支持添加和管理多个使用者呼号，并可设置一个主要呼号用于日常操作。
  

## 安装与依赖

在运行此程序前，请确保已安装 Python 3.x 环境。

1. **安装必要的库**:
  本项目依赖于多个第三方库。您可以通过 pip 来安装它们：
  
  ```
  pip install PyQt5 adif-io qrcode reportlab Pillow PyMuPDF pyserial
  ```
  
2. **字体和图标文件**:
  为了保证程序界面和生成文件的显示效果，请确保以下文件与主程序 `main.py` 位于同一目录下：
  
  - `MapleMonoNL-Regular.ttf` (用于显示英文字符)
    
  - `Cinese.ttf` (用于显示中文字符)
    
  - `icon.ico` (用作程序的图标)
    

## 如何运行

直接通过 Python 解释器运行主脚本即可启动程序：

```
python main.py
```

程序启动后会自动在当前目录下创建 `database/` 和 `labels/` 文件夹，用于存放数据库文件和生成的标签图片。

## 打包为可执行文件 (.exe)

如果您希望将此程序打包成一个在 Windows 上无需安装 Python 环境即可运行的独立 `.exe` 文件，请遵循以下步骤：

1. **安装 PyInstaller**:
  PyInstaller 是一个能将 Python 程序打包成独立可执行文件的工具。
  
  ```
  pip install pyinstaller
  ```
  
2. **下载 `build.spec` 文件**:
  为了精确控制打包过程，特别是为了包含字体和图标等非代码文件，我们需要一个 `.spec` 文件。请下载项目中名为 `build.spec` 的文件，并放置在代码同级目录下
  
3. **执行打包命令**:
  打开命令行，切换到项目根目录（即 `main.py` 和 `build.spec` 所在的目录），然后运行：
  
  ```
  pyinstaller build.spec
  ```
  
4. **找到可执行文件**:
  打包过程结束后，您会在项目根目录下看到一个名为 `dist` 的新文件夹。进入该文件夹，即可找到 `QSL_Card_Manager.exe`，这就是最终的可执行程序。
